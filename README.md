A Claude skill that takes your resume and a job description and outputs a tailored, one-page ATS friendly resume as a .docx and .pdf.

What it does

Reads your full resume and the target job description
Rewrites and tailors bullets, summary, and skills to match the role
Trims or reframes experience that doesn't fit the JD
Fills exactly one page — checks visually before delivering
Outputs both a Word doc and PDF, ready to submit


How to install

Download resume-writer.skill
In Claude.ai, go to Settings → Skills
Click Install skill and upload the file


How to use
Upload your resume + paste the job description in any Claude conversation. The skill triggers automatically and delivers both files.
